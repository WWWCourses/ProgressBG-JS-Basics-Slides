Overview of REST APIs

Overview of REST APIs

REST (Representational State Transfer) is an architectural style used for designing networked applications.
RESTful APIs are widely used in web and mobile applications for their simplicity and scalability.
REST APIs use standard HTTP methods for CRUD (Create, Read, Update, Delete) operations.
This makes it easier to integrate with various clients and platforms.

CRUD Operations and Their REST Equivalent

CRUD Operations and Their REST Equivalent

CRUD Operations and Their REST Equivalent

DB CRUD Operation HTTP Method Server Endpoint Description
Create POST /users Creates a new resource
Read GET /users/1 Retrieves a specific resource
Read (List) GET /users Retrieves a list of resources
Update (Full) PUT /users/1 Updates an entire resource
Update (Partial) PATCH /users/1 Updates part of a resource
Delete DELETE /users/1 Deletes a specific resource

GET Request Example

We want to send a GET request to the jsonplaceholder API to retrieve a post with ID 1 and to log the response data in JSON format.
Example with fetch() API

                fetch('https://jsonplaceholder.typicode.com/posts/1')
                    .then(response => response.json())
                    .then(data => console.log(data))
                    .catch(error => console.error('Error:', error));
Example with async/await

                async function getPost() {
                    try {
                        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1');
                        const data = await response.json();
                        console.log(data);
                    } catch (error) {
                        console.error('Error:', error);
                    }
                }

                getPost();

POST Request Example

We want to send a POST request to the JSONPlaceholder API to create a new post with a title, body, and user ID, then log the response data in JSON format

                fetch('https://jsonplaceholder.typicode.com/posts', {
                    method: 'POST',
                    body: JSON.stringify({
                        title: 'New Post',
                        body: 'Post content',
                        userId: 1,
                    }),
                    headers: {
                        'Content-type': 'application/json; charset=UTF-8',
                    },
                })
                    .then((response) => response.json())
                    .then((data) => console.log(data))
                    .catch(error => console.error('Error:', error));
Example with async/await

                async function createPost() {
                    try {
                        const response = await fetch('https://jsonplaceholder.typicode.com/posts', {
                            method: 'POST',
                            body: JSON.stringify({
                                title: 'New Post',
                                body: 'Post content',
                                userId: 1,
                            }),
                            headers: {
                                'Content-type': 'application/json; charset=UTF-8',
                            },
                        });
                        const data = await response.json();
                        console.log(data);
                    } catch (error) {
                        console.error('Error:', error);
                    }
                }

                createPost();

DELETE Request Example

We want to send a DELETE request to the JSONPlaceholder API to remove an existing post by its ID, then log the response status

                fetch('https://jsonplaceholder.typicode.com/posts/1', {
                    method: 'DELETE',
                })
                    .then((response) => console.log('Deleted with status:', response.status))
                    .catch(error => console.error('Error:', error));
Example with async/await

                async function deletePost() {
                    try {
                        const response = await fetch('https://jsonplaceholder.typicode.com/posts/1', {
                            method: 'DELETE',
                        });
                        console.log('Deleted with status:', response.status);
                    } catch (error) {
                        console.error('Error:', error);
                    }
                }

                deletePost();

Testing REST APIs

Testing REST APIs

Overiew

Testing REST APIs ensures they function correctly by verifying request-response interactions.
API testing helps check data consistency, authentication, error handling, and performance.

Tools for REST API Testing

RestMan: Lightweight Chrome extension for quick API requests.
cURL: Command-line tool for sending HTTP requests.
Postman: Feature-rich API client for manual and automated testing.
Swagger UI: Interactive documentation and testing tool.
Insomnia: Modern API testing tool with GraphQL support.
